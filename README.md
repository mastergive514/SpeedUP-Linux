# Speed UP
Speed up is a free python file, can speed up your pc!



## Instalation
**Download files in your pc**

```$ git clone https://github.com/mastergive514/SpeedUP-Linux.git```

**go to Directory of files**

```$ cd SpeedUP-Linux```

**Run SpeedUP**

```$ python Speedup.py```

## Support 

Only **Linux** x64/x32!
